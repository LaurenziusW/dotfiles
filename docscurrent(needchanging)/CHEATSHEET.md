# UKE Cheat Sheet

## Window Management (Cmd/Alt)
```
Mod + H/J/K/L     Focus left/down/up/right
Mod + 1-0         Switch workspace
Mod + [/]         Prev/next workspace
Mod + Tab         Recent workspace
```

## Move Windows (Mod+Shift)
```
Mod+Shift + H/J/K/L    Move window
Mod+Shift + 1-0        Move to workspace
Mod+Shift + F          Fullscreen
Mod+Shift + Space      Float
```

## Resize (Alt+Shift)
```
Alt+Shift + H/J/K/L    Resize
```

## Launchers (Cmd+Alt / Super)
```
Ter + B    Browser
Ter + O    Obsidian
Ter + C    VS Code
Ter + M    Spotify
Ter + F    Files
```

## Bunches (Cmd+Ctrl / Super+Ctrl)
```
Bunch + 1  Study
Bunch + 2  Guitar
Bunch + 3  Coding
Bunch + 4  Email
Bunch + 5  Reading
```

## Caps Lock Layer
```
Caps + H/J/K/L    Arrows
Caps + W/B        Word nav
Caps + U/D        Page up/down
Tap Caps          Escape
```

## Tmux (Ctrl+A)
```
C-a + \ / -       Split H/V
C-a + h/j/k/l     Navigate panes
C-a + c           New window
C-a + 1-5         Switch window
```

## WezTerm (Alt)
```
Alt + t/w         New/close tab
Alt + 1-5         Switch tab
Alt + \ / -       Split
Alt + Shift+hjkl  Navigate panes
```

## CLI
```
uke gen           Generate configs
uke reload        Reload WM
uke edit          Edit registry
uke status        Status
uke-bunch list    List bunches
uke-gather        Organize windows
```
